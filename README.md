# ضيافتكم (Diyafatkum)

منصة رقمية كويتية تربط العملاء بمقدمي خدمات الضيافة النسائية بطريقة احترافية.

## المرحلة الأولى

الخدمات المتوفرة:

- استقبال
- عشاء

## التقنيات

- Frontend: React + TypeScript
- Backend: Supabase
- Database: PostgreSQL
- Authentication: Supabase Auth
- Storage: Supabase Storage

## أهداف المرحلة الأولى

- إدارة العملاء
- إدارة مقدمي الخدمة
- إدارة الباقات
- إدارة الطلبات
- إدارة الشكاوى
- إدارة التقييمات
