# معمارية النظام (System Architecture)

## المستخدمون

1. العميل
2. مقدم الخدمة
3. المدير
4. مالك التطبيق

---

## الوحدات الرئيسية

### المصادقة
- تسجيل الدخول
- إنشاء الحساب
- استعادة كلمة المرور

### العملاء
- الملف الشخصي
- العناوين
- الطلبات
- الرسائل
- الشكاوى
- التقييمات

### مقدم الخدمة
- بيانات النشاط
- التراخيص
- الخدمات
- الباقات
- الطلبات
- الرسائل
- التقييمات

### المدير
- اعتماد مقدمي الخدمة
- مراجعة الشكاوى
- إدارة المستخدمين
- التقارير

### مالك التطبيق
- لوحة المؤشرات
- متابعة أداء المدير
- الإحصاءات
- الإعدادات العامة

---

## دورة الطلب

العميل
↓
اختيار الخدمة
↓
اختيار الباقة
↓
اختيار العنوان
↓
إرسال الطلب
↓
قبول أو رفض مقدم الخدمة
↓
تنفيذ الخدمة
↓
إغلاق الطلب
↓
التقييم أو الشكوى
